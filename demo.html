Grading Policy

Item Percentage Description
In-Class Participation & In-Class Activities 5% Actively speaking/typing, showing code for review, working with Copilot actively and responsibly.
Assignments 30% Task 1 (Intro to Python), Task 2 (CLI Data Analyzer), Task 3 (Clone + Extend + API + Improvements), Task 4 (Group Project + Git Workflow).
Project Code Quality & GitHub Repository 35% Final project organization, documentation, meaningful commit history, README clarity.
In-Class Project Presentation 15% Clear explanation of project using comments and documentation, demonstration of understanding, ability to answer how/why decisions were made.
Final Exam (In-Person) 25% Short coding + reasoning questions assessing: Understanding of Python, ability to explain code, light debugging. No Copilot during the exam.
