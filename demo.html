Grading Policy

Item Percentage Description
Participation & In-Class Activities 10% Engagement, coding practice, working with Copilot actively and responsibly.
Task Assignments 20% Task 1 (Python basics), Task 2 (Project Build), Task 3 (Clone + Extend + API + Improvements), Task 4 (Collaborative Enhancement + Git Workflow).
Project Code Quality & GitHub Repository 35% Final project organization, documentation, meaningful commit history, README clarity.
In-Class Project Presentation 15% Clear explanation of project using comments and documentation, demonstration of understanding, ability to answer how/why decisions were made.
Final Exam (In-Person) 25% Short coding + reasoning questions assessment: understanding of Python, ability to explain code, light debugging, limited copilot usage allowed on select questions.
